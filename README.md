# Diagnóstico e Priorização de Investimentos em Saneamento – SP (2022)

## Contexto
Este projeto utiliza dados oficiais do Sistema Nacional de Informações sobre Saneamento (SNIS)
para analisar a situação do saneamento básico nos municípios do estado de São Paulo.

## Objetivo
Construir um índice composto (0–100) para avaliar o desempenho dos municípios em saneamento
e apoiar a priorização de investimentos públicos com base em critérios técnicos.

## Fonte de Dados
- Sistema Nacional de Informações sobre Saneamento (SNIS) – Ministério das Cidades

## Metodologia
- Seleção de indicadores de cobertura, eficiência e infraestrutura
- Normalização Min-Max
- Construção de índice ponderado
- Geração de ranking e métrica de risco ajustada por investimento municipal

## Principais Insights
- Municípios com menor índice de saneamento apresentam, majoritariamente, ausência de investimento municipal.
- A combinação de baixo índice e baixo investimento resulta em alto risco de perpetuação de problemas estruturais.
- Foram identificados municípios com investimento relevante, porém sem melhora proporcional no índice,
  indicando possíveis ineficiências na alocação de recursos.

## Recomendações
- Priorizar municípios com baixo índice de saneamento e ausência de investimento municipal.
- Avaliar a eficiência dos investimentos realizados em municípios com baixo desempenho.
- Utilizar o índice como ferramenta de apoio à decisão para políticas públicas.

## Limitações
- Dependência do reporte municipal ao SNIS.
- Análise restrita ao ano de 2022.
